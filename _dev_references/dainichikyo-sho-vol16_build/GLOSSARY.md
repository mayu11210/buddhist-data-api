# 大日経疏 巻第十六 現代語訳 共通用語集（全バッチ統一）

## 固有名・術語（統一表記）
- 毘盧遮那＝大日如来〔法身仏〕。「遮那」も毘盧遮那。
- 金剛手・執金剛・秘密主・持金剛＝同一の大菩薩（秘密を司る）。「大我」も仏の別名。
- 漫荼羅＝曼荼羅〔まんだら〕。原文の「漫荼羅」表記は訳文では「曼荼羅」に統一してよい。
- 阿闍梨〔あじゃり・師〕／持誦者〔じゅそうしゃ・真言を唱え修する行者〕。
- 浄菩提心〔じょうぼだいしん〕は訳語化せずそのまま用い、必要に応じ〔さとりを求める清らかな心〕と割注。
- 三昧耶〔さんまや〕／三昧〔さんまい・精神統一〕／等至〔とうじ・三昧の名。三世の仏が等しく至る道〕。
- 悉地〔しっじ・成就〕。潅頂〔かんじょう〕。護摩。
- 種子〔しゅじ・本尊を表す梵字一字〕／印〔いん・印契・手の象徴形〕／真言〔しんごん〕。
- 五事＝印・色・尊位・住・三昧（曼荼羅を解する五つの要件）。
- 五大＝地・水・火・風・空。字観では阿〔地〕・嚩（ば）〔水〕・囉（ら）〔火〕・訶（か）〔風〕・（空は点 欠字）。※梵字音は割注で（ )ではなく〔 〕に。例：嚩字〔ば字〕。
- 本不生〔ほんぷしょう・もともと生じたものでないこと。阿字の義〕。
- 大悲胎蔵〔だいひたいぞう〕曼荼羅／悲生壇〔ひしょうだん〕。
- 三部＝仏部・蓮華部・金剛部。釈迦部・菩薩部も登場。
- 商佉〔しょうか・法螺貝〕。如意珠〔にょいしゅ〕＝真陀末尼・摩尼。
- 観音＝観自在＝観吾。馬頭〔めず〕＝訶耶〔かや〕。
- 諸尊名（仏頂・諸天・諸菩薩）は原語のまま、必要なら短い割注。
- 質多〔しった・citta・心〕。阿頼耶〔あらや・蔵・心の奥底の貯蔵庫〕。
- 満奴〔まぬ・我〕・末那〔まな〕＝外道の見の名。
- 十二字真言王＝身体各部に配する十二の真言の字。
- 方・円・三角・半月＝曼荼羅の形（地水火風に配当）。

## 文体ルール
- 一文を長くしすぎず、原文の論理（いわく／謂わゆる／故に／当に〜べし）を平明に。
- 儀軌の作法描写（「〜を作せ」「〜を置け」）は「〜を作る」「〜を置く」と読み手に分かる現代文に。
- 経文の引用・問答は地の文として自然に訳す。
- 「更に問え」「具に問え」等の原注的指示は〔さらに問え〕等として残す。
- 原文にある割注（（ ）内の語釈）は〔 〕に変換して本文へ自然に織り込むか割注として残す。
