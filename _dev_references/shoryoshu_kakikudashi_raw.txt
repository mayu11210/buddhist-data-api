# 全文書き下し 遍照発揮性霊集（Kindle）
# PowerToys Text Extractor 抽出テキスト収集ファイル
#
# 使い方：
#   1. Kindle for PC で対象ページを開く
#   2. PowerToys のテキスト抽出（Win+Shift+T）でページ全体を選択
#   3. クリップボードにコピーされたテキストをこのファイルに貼り付ける
#   4. ページ区切りとして「=== ページ N ===」を手動で挿入する（任意）
#      例：=== 巻第一 遊山慕仙詩 p.1 ===
#   5. fix_columns.py を実行して列順を修正 → shoryoshu_kakikudashi.txt に出力
#
# 注意事項：
#   - PowerToys は縦書きテキストを「左から右」の列順で抽出する
#   - 日本語縦書きの正しい読み順は「右から左」なので、列が逆順になる
#   - fix_columns.py がこれを自動修正する
#   - このファイルは原文OCR（shoryoshu_raw.txt）の校正照合元として使用する
#   - shoryoshu.json の内容を修正対象とする際もこのデータを参照する
#
# 列区切り検出ロジック（fix_columns.py 参照）：
#   - 空行 → 列の区切りとして扱う
#   - 「===」で始まる行 → ページ区切りとして扱う（逆順処理はページ内のみ）
#
