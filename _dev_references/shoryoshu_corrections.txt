============================================================

■ 修正確認済み（PDF画像目視・本セッション確認）

修正候補　：覚誰非理 → 鳧鶴誰非理

元テキスト：出鎭狄之才狼之歡顔之遠阻

修正候補　：狄 → 豺（OCR誤読）、才 → （欠字・OCR混入）、
           歡顔 → 龍顔（歡 → 龍、OCR誤読）
補足　　　：「出鎭狄之才狼之歡顔」は複数行が合流した重度OCR乱れ。

元テキスト：貧道与君遠如知

補足　　　：前後の「白雲之人」が欠落している可能性あり。目視確認推奨。

元テキスト：情登無日歡聚、抽拔君義以充遠邊之霧

           遠邊之霧 → 辺霧之頤（遠 → 頤 等、重度誤読）
補足　　　：「情登無日歡聚」部分は書き下しとの対応不明。目視確認要。

元テキスト：日本廓城三州天邊

元テキスト：之更奥日黙念天皇棣以怒幾悉按将

書き下し　：「就の中に陸奥最も柔げ難し」「天皇赫怒して」「幾たびか剣を按しはる」
修正候補　：之更奥 → 就中陸奥（之 → 就、更 → 陸、日 → （欠字・OCR混入））
           天皇棣以怒 → 天皇赫怒（棣以 → 赫）
           幾悉按将 → 幾按剣（悉 → （欠字）、将 → 剣）

元テキスト：毛中争覇馳謀往在帝伐

【ページ36】喜雨歌（散文部分）
------------------------------------------------------------

■ 修正確認済み（書き下し照合）

元テキスト：「哀哉末世讃元霊不屑。」「寃三界夢愛地永元霊不屑。」原身。
照合結果　：哀哀末世諸元元。聾瞽不屑聖者之言。久眠三界夢。永愛四蛇之原。身口意行十悪。
書き下し　：「哀哀たり末世の諸の元元。聾瞽にして聖者の言を屑にせず。
           久しく無明の酒に酔うて本覚の源を知らず。長く三界の夢に眠りて、
           永く四蛇の原を愛す。身と口と心とに十悪を行ず。」
修正候補　：讃元霊 → 諸元元（讃 → 諸、霊 → 元 OCR誤読）
           寃 → 久眠（相当する漢字要確認）、愛地 → 四蛇之原（地 → 四蛇之 重度誤読）
補足　　　：OCR列の合流が著しく、複数文が一行に混合している。

元テキスト：蕩遊佇迷。与口行十忠不孝罪業繁擾。
照合結果　：蕩逸昏迷。営口腹。不忠不孝。罪業繁。
書き下し　：「蕩逸昏迷にして口腹を営む。...不忠不孝にして罪業繁し。」
修正候補　：遊 → 逸（OCR誤読）、佇 → 昏（OCR誤読）
           与 → 営（OCR誤読）、行 → 腹（OCR誤読）

元テキスト：因果戯身。誓口腹生之之笑而哭打東打北。
照合結果　：撥因果。生死笑而哭。打東打北。
書き下し　：「因果を撥して罪福を無にす。生れ之き死にて之いて笑いて而も哭す。東に打たれ北に打たるること」
修正候補　：戯 → 撥（OCR誤読）
           生之之 → 生死（之之 → 死 OCR誤読）
           打東打北 ✓（正確）

元テキスト：聖者言久酔無明酒不知本覚源
照合結果　：聾瞽不屑聖者言。久酔無明酒不知本覚源。
書き下し　：「聾瞽にして聖者の言を屑にせず。久しく無明の酒に酔うて本覚の源を知らず」
修正候補　：聖者言の前に「聾瞽不屑」が欠落（OCR行の合流による欠落の可能性）
           久酔無明酒不知本覚源 ✓（正確）
