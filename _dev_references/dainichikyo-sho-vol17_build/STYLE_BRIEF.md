# 大日経疏 巻第十七 現代語訳ブリーフ（butten-yasashii-yaku 準拠）

漢文仏典の書き下し文（訓読文）を、一般読者が一読で意味を取れる**平易な現代語**に訳す。
同時に原文の漢語・術語・特徴句を**割注〔 〕で多めに残す**。読みやすさと原典参照性の両立が目的。

## 最重要原則：「書き下しに注を付けただけ」にしない
最悪の失敗は、書き下し文をほぼそのまま本文に置き、難語に注を添えただけで済ますこと。
**本文は文を組み替えて現代語として再構成する。** 長文は短く切り、主語・目的語を補い、
「たとえば」「そして」「それで」等の接続を補う。文末は「〜である」「〜という」「〜だろうか」。
「〜なり」「〜したまえり」等の文語語尾は本文では使わない（経文・偈・真言の引用で原文の
響きを残すときだけ例外可）。原文の漢語・術語はその直後の〔 〕に退避させる。

### 避ける例（書き下し＋注のまま）
> 異生〔いしょう・迷いの生をうけた者〕羝羊〔牡羊〕の心とは、すなわちこれ凡夫が…なり。

### 目指す例（本文を平易に組み替え、原語を注へ）
> これは人の心のいちばん低い段階〔住心の第一〕である。聖者にはほど遠い普通の人〔異生・凡夫〕が、
> まるで牡羊〔羝羊〕のように本能のまま生きている心をいう。

## 文体規定
1. 本文は平易な現代語。文末も現代語。
2. 原文の漢語・術語・特徴句は割注〔 〕で多めに残す（注は「語義の言い換え」でなく「原文語そのもの」を入れる）。
3. 固有名詞・難読語・教学術語は〔読み・短い意味〕でよい。注は学術脚注のように長くしない。
4. 経名のみ示す。T番号・訳者名は書かない（例『大日経』『金剛頂経』『涅槃経』『金光明経』『法華経』『仏性論』『宝蔵経』）。
5. Sanskrit 復元は基本教学用語のみ最小限（本不生・三摩地・等引等）。
6. 真言・陀羅尼の音写は〔読み＋教義的意味〕を添える（例 阿字〔ア／一切法本不生〕、嚩字〔バ／水・無縛〕、囉字〔ラ／火〕、訶字〔カ／風〕、佉字〔キャ／空〕）。
7. 偈・頌は対句の構造を保ちつつ訳す。散文に均さない。
8. 密教教学用語は原語維持（三密・阿字・本不生・即身成仏・曼荼羅・三昧耶 等）。

## 厳守の表記ルール（機械検証あり）
- **全角括弧（）・割注〔 〕を使う。本文・訳文に半角 ( ) を一切混ぜない。**
- 画像位置標識 `〔図・…〕` が kakikudashi に含まれる段は、**その〔図・…〕標識を訳文にもそのまま残し**、前後の地の文だけを現代語訳する（標識内は改変しない）。
- 訳は段ごとに完訳する。未訳・省略・「以下略」は不可。

## 巻第十七 共通 GLOSSARY（表記統一）
- 毘盧遮那〔びるしゃな・大日如来〕／遮那 … 「大日如来〔毘盧遮那〕」で統一
- 金剛手〔こんごうしゅ・執金剛・秘密主〕…質問者の菩薩
- 阿闍梨〔あじゃり・密教の師〕／弟子
- 八印〔はちいん〕＝秘密八印（第一〜第八印）。各印に印相・漫荼羅・真言がある
- 三補吒〔さんぽた〕…合掌の一種／虚心合掌
- 漫荼羅〔まんだら〕（原文表記「漫荼羅」を割注に温存しつつ本文は「曼荼羅」可、ただし本巻は「漫荼羅」で統一）
- 真言〔しんごん〕／種子〔しゅじ・種子字〕／悉曇〔しったん・梵字〕
- 阿字〔あじ／一切法本不生〕…本不生＝もともと生じたものでない真実
- 嚩字〔ばじ／水・無縛〕囉字〔らじ／火〕訶字〔かじ／風〕佉字〔きゃじ／空〕欠字〔けん／空〕
- 三平等〔さんびょうどう〕＝身は印・語は真言・心は本尊の三つが等しいこと（三密）
- 持明禁戒〔じみょうごんかい〕…真言行者の戒。尸羅〔しら・本性戒〕と没栗多〔もつりた・時限ある願戒〕
- 落叉〔らくしゃ〕…浅略には十万遍、ここでは「見〔真実を見ること〕」「探（的を射る）」の義
- 等引〔とういん／三摩呬多・さんまきた〕…定慧が等しく合一する境地
- 六月持誦法〔ろくがつじじゅほう〕…第一月=金剛・阿字・黄、第二月=水輪・嚩字・白、第三月=火輪・囉字・赤、第四月=風輪・訶字・黒、第五月=金水・阿嚩、第六月=風火・訶羅
- 布字〔ふじ〕…身体の各部位に梵字を配する観法
- 受方便学処〔じゅほうべんがくしょ〕…菩薩戒（学処）を受けること
- 菩薩戒／十善戒〔不殺生・不与取・不邪婬・不妄語・不悪口・不両舌・不綺語・不貪・不瞋・不邪見〕
- 偸蘭遮〔ちゅうらんじゃ〕…重罪に次ぐ罪／四重禁・十重禁
- 三帰〔さんき・三宝への帰依〕／十無尽蔵〔じゅうむじんぞう〕
- 索多〔さくた〕＝薩埵〔さった・有情・菩提薩埵〕…「著（執着）」「有情」の義
- 瑜伽〔ゆが〕／瑜祇〔ゆぎ・相応〕／瑜伽座〔大因陀羅坐・四方金剛輪の坐〕
- 阿伽羅〔あから・阿字は一切字に遍ず〕／阿刹羅〔根本字・男声・慧〕／哩比〔りび・増加字・女声・定〕
