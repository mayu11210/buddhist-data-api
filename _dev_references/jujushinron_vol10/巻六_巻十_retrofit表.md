## 3. 縦糸（三本のテーマ軸）

1. **本不生軸**：C20 阿字本不生 → D13 無量の不生。「生じない」一点が、あらゆる教えを貫く無量の根源へ。
2. **即身成仏軸**：C21 一生成仏 → D03 如実知自心 ＋ D08 法仏平等の身口意。さとりの定義と身口意の一体化で成仏を根拠づけ（D08 は genten「即是法佛平等身口意」＝三密ではなく身口意で確定）。
3. **十住心総括軸**：全九住心 → D05 心続生／D01 到達点。竪十重・横塵数（D04）が骨格。宝鑰頌「九種住心無自性」が凝縮。

---

## 4. 秘蔵宝鑰（略本）による裏づけ（2026-05-31・SAT 直接取得）

- **十住心総括軸**：宝鑰頌「九種住心無自性　轉深轉妙皆是因」＝顕九住心→密への深化を一聯に圧縮。
- **本不生軸（C20↔D13）**：宝鑰が引く金剛頂経偈「我覺本不生」が D13 と直結。
- **即身成仏軸（C21↔D03/D08）**：宝鑰頌「萬徳自性輪圓足　一生得證莊嚴仁」＝一生成仏の凝縮句。
- **四智→五智（D07）**：宝鑰が菩提心論「五方佛各表一智…」を引いて五智五方仏を保持。広↔略ともに五智で対応明瞭。

## 5. 華厳の字門連関の解決（SAT 確認）

D06 字門 ↔ 華厳の連関は別巻を探すまでもなく**巻十 D06 本文に内蔵**。空海自身が問答で「華嚴・般若・涅槃の四十二字（阿羅波遮那＝入法界品 善知衆藝童子の字門）は末・浅略、大日経の四十九字（悉曇）が本・深秘」と顕密対弁する。D06 は字輪列挙＋顕密対弁問答の二部構成。
