## ★訂正（2026-05-31・buddhist-jujushinron-workshop の正本と照合した結果）

本フォルダの各資料には、巻十の正本（`buddhist-jujushinron-workshop/data/kukai/jujushinron.json`・原典は SAT T2425）と照合した結果、以下の**誤りが判明している**。正本が優先。

- **D02（曼荼羅）の判定は誤り**：本資料は「金剛頂十八会は本文になし・胎蔵海会／金剛海会の二会のみ・底本は海會」としているが、**正本の巻十 genten は三会**＝「**胎藏海會曼荼羅。金剛界會曼荼羅。金剛頂十八會曼荼羅是也**」。すなわち金剛頂十八会は**本文にある**し、二つ目は金剛**界**会。元の motif 候補ドラフトの D02（三会）が正しく、当方の「除外」訂正が誤りだった（筑摩の要文＝略形に引かれた）。
- **D01（自心の数量）**：正本 genten は「如實證悟**自身之數量**」。一方、正本の**書き下しは「身の数量」（自 脱落）**で原典と不一致。要検討（底本準拠で残すか、自を補うか）。
- D08（法仏平等の身口意・≠三密）・D13（末尾欠）・D06（四十二字/四十九字）・D12（菩提場経）は正本と整合。

以後、巻十の原典・書き下し・現代語訳の正本は **buddhist-jujushinron-workshop 側の jujushinron.json**（全10巻・現代語訳 新訳調統一済）であり、本フォルダはあくまで motif 前段の参考に留める。

## 経緯と到達点

別ワークスペースで『秘密曼荼羅十住心論』巻十「秘密荘厳住心」の motif 候補 D01–D13 を選定し、各箇所の **原典（genten）・書き下し（kk）・現代語訳（gendai）** を生成。原典は **SAT 大正蔵 2018（T2425 巻十）で全件行レベル照合**して確定した。

- 主な確定・訂正：D01「自身之數量」（旧「身の数量」誤）／D02「金剛頂十八會は本文になし」→除外（注釈書由来）／D08「法佛平等**身口意**」（旧「三密」誤）・「真言門」／D13 各乗の不生義の全列挙と**末尾欠（極無自性乗の段で本文中断・未完）**。
- 重要事実：**CBETA は大正蔵 vol.77〔諸宗部・日本撰述〕を非収録**のため、T2425（十住心論）・T2426（秘蔵宝鑰）は **SAT が唯一の正典出所**。
